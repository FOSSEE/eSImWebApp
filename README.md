##eSimWebApp

##Pre-requisites
1. Node.js (v5.3.0)
2. Ngspice


##How to run ?
1. Run `node app.js`

2. Open browser and go to http://localhost:4000

##Setting environment variable 

####For test server

    `export NODE_ENV="testing"`

####For development

    `export NODE_ENV="development"`

####For production server

    `export NODE_ENV="production"`
